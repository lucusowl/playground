# Linux 초기 설정

CLI환경에서의 작업으로 서술[^1]

## 작업

1. (선택) 시스템 정보 확인
2. 관리자계정 활성화
3. 네트워크 설정
4. 패키지 업데이트
5. (선택) 시스템 언어 설정
6. 시스템 시간 동기화
7. SSH 원격 연결 활성화
8. (선택) 기타 편의 설정
   1. 키보드 레이아웃
   2. 시스템 로케일
   3. 응용 계층 서비스 설정
